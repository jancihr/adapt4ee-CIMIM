adapt4ee-CIMIM
==============

**adapt4ee-CIMIM** (**Common Information Model Interface Module**) project provides reference implementation of storage, inference and query services for the [adapt4ee-CIM](https://github.com/jancihr/adapt4ee-CIM) (Common Information Model). Module provides REST-like services for XML data exchange and SPARQL interface for RDF instances and semantic model.

For more information about the adapt4ee CIM model and CIMIM services, follow the [adapt4ee](http://www.adapt4ee.eu) project page.
